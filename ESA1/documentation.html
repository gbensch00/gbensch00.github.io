Verwendete Tools und Frameworks:

Die webseite inklusive der Tabelle ist komplett mit html, css und javascript erstellt worden. Der Graph wurde mit GoJS erstellt. Das Design habe ich hierbei so schlicht und simpel gehalten wie möglich. Hochgeladen habe ich die Webseite über Github.

Implementierung:

Die Wortgeneration habe ich implementiert, indem ich alle korrekten Wörter in ein Array und inkorrekte Wörter in ein anderes Array gepackt habe, und ein zufälliges Wort aus dem Array genommen wird wenn man auf den entsprechenden Button drückt. Zum testen wird geprüft, ob das Wort in dem Array ist. Leider fehlen bei dieser Implementation natürlich die zwischenschritte, wodurch sich der Graph und die Tabelle noch nicht richtig animieren lässt, aber eine andere Implementation habe ich noch nicht geschafft, da ich vorher keine Erfahrung mit Web Development und vor allem Frameworks hatte.
